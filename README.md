## Features

- Open local MIDI files for piano practice, with tempo adjustment, loop practice, left-hand/right-hand modes, and customizable AI assistance
- Upload MP3 files and automatically transcribe them into high-quality MIDI files through [Transkun](https://github.com/Yujia-Yan/Transkun)
